# Java Socket Programming — TCP Client/Server

A Computer Networks mini-project built with **Java TCP sockets**:
- **Multi-client server** (thread per connection)
- **CLI client**
- Supports two actions:
  - **Square** of a number
  - **File request** (server sends file bytes, client saves a copy)

## Features
- **TCP server** on port `8888`
- **Multiple clients** supported concurrently
- Simple request/response interaction from the terminal
